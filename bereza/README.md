# БЕРЕЗА — особистий медіа-проєкт

Прототип сайту в стилі розслідувального медіа. Чиста статика без фреймворків.

## Структура

```
bereza/
├── index.html              головна
├── pages/
│   ├── article.html        приклад сторінки матеріалу
│   └── about.html          про автора
├── css/
│   ├── reset.css           CSS reset
│   ├── tokens.css          змінні (кольори, шрифти, відступи)
│   ├── base.css            базова типографіка, фон, утиліти
│   ├── layout.css          topbar / masthead / nav / footer
│   └── components.css      hero, картки, відео, цитата, форми
├── js/
│   ├── main.js             точка входу: завантажує дані
│   └── render.js           рендер карток із JSON
├── data/
│   └── content.json        контент сайту (статті, відео, цитати)
└── README.md
```

## Як подивитися

### Локально через сервер (рекомендовано)

З папки `bereza/`:

```bash
python3 -m http.server 8000
# або:
npx serve
```

Далі відкрити `http://localhost:8000`.

### Через подвійний клік на index.html

Теж працюватиме — у головній сторінці є вбудований об'єкт
`window.__BEREZA_DATA__` як фолбек на випадок, якщо браузер
заблокує `fetch()` для `file://`. На сервері використовується
JSON із `data/content.json`.

## Як редагувати контент

Усі матеріали головної живуть у `data/content.json`:

- `ticker` — рядок-біжучка нагорі
- `hero` — головний матеріал
- `heroSide` — топ-4 у бічній колонці
- `columns` — три авторські колонки
- `videos` — три відео для секції YouTube
- `quote` — цитата (підтримує `<em>` для виділення)
- `latest` — сітка свіжих матеріалів
- `channels` — соцмережі в підвалі секції підписки

> ⚠️ Коли правите JSON — продублюйте зміни і у вбудованому
> `window.__BEREZA_DATA__` всередині `index.html`, інакше при
> відкритті `file://` буде стара версія.

## Дизайн-токени

Усі кольори, шрифти і відступи — у `css/tokens.css`. Щоб
змінити фірмовий акцент, відредагуйте `--accent` (зараз
кислотний лайм `#d4ff3a`) і `--accent-2` (тривожний
помаранчево-червоний `#ff4422`).

## Технології

- HTML5 + CSS3 (Grid, custom properties, container-queries-ready)
- Vanilla JS (без бандлерів)
- Шрифти: Unbounded (заголовки), Spectral (текст), JetBrains Mono (метадані)

## Що ще можна додати

- Сторінки рубрик (колонки, розслідування, відео) як шаблон списку
- Бекенд або CMS (Strapi / Decap CMS / просто Markdown + 11ty)
- RSS-стрічка
- Темна/світла версії — токени дозволяють перемкнути одним свічем
- Sitemap, OG-зображення, метатеги для соцмереж
